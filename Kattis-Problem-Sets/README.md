# Kattis-Problem-Sets <img src="https://ia804606.us.archive.org/23/items/medium_202201/medium.png" width="36px"/>

GitHub.io page [here](https://alimuhammadasad.github.io/Kattis-Problem-Sets/). View source [here]().

This is the webpage I created for a research project I was working on. This webpage contains the problem sets that to be used in the course once it commences. The webpage is purely hosted using GitHub Pages, with a Jekyll Theme. The source code is just a simple `.md` file, however, the `.md` file contains HTML tags, HTML structure along with some CSS for styling and JavaScript for minimal interactivity.

Problem sets created from <a href="https://open.kattis.com/">Kattis Archives</a>.

    function pCount(){
        var tables = document.querySelectorAll('#container table');
        var ps = 0;
        tables.forEach(function(table){
            ps += table.rows.length;
        });
        
        document.getElementById('total').textContent = ps - 15;
    }
    pCount();
